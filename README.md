# 💰 Expense Tracker – Full Stack MERN Application

A production-ready **Expense Tracker Web Application** built using modern full-stack technologies. This application enables users to efficiently manage their income and expenses with secure authentication, interactive dashboards, and real-time data visualization.

---

## 🚀 Project Overview

The Expense Tracker is a **full-stack web application** designed to help users track their financial activities. It provides detailed insights into income and expenses through dynamic charts and reports.

The system includes:

* 🔐 Secure User Authentication (Login/Register)
* 📊 Interactive Dashboard with Financial Insights
* 💸 Income & Expense Management
* 📅 Weekly & Monthly Analysis
* 📈 Data Visualization using Charts

## 🛠️ Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* React Router DOM
* Axios
* Framer Motion (animations)
* Recharts (charts & graphs)
* React Toastify (notifications)

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* bcrypt (password hashing)
